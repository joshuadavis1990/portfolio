HTML and CSS Basics

HTML and CSS are two of the most important tools used in web development. HTML, or HyperText Markup Language, is used to create the structure and content of web pages, while CSS, or Cascading Style Sheets, is used to style and layout those pages.

HTML is made up of a series of tags, which are used to mark up the different parts of a web page. These tags are enclosed in angle brackets, and most have an opening and closing tag. The content between the opening and closing tags is what will be displayed on the web page. Some of the most common HTML tags include:

  • <h1> - <h6> for headings
  • <p> for paragraphs
  • <a> for links
  • <img> for images
  • <ul> and <li> for unordered lists
  • <ol> and <li> for ordered lists

CSS is used to style and layout web pages, and is separate from the HTML markup. CSS styles are applied to HTML elements using selectors, and can be written in a separate CSS file or directly in the HTML using the <style> tag.

        p {
          font-size: 16px;
          color: #333;
          line-height: 1.5;
        }

This rule applies to all <p> tags on the page, setting their font size to 16 pixels, color to #333 (a dark gray), and line height to 1.5 times the font size.

Some common CSS properties include:

  • font-family
  • font-size
  • color
  • background-color
  • padding
  • margin
  • border
  • display
  • float

HTML and CSS are essential tools for web developers. HTML provides the structure and content of web pages, while CSS provides the styling and layout. By mastering these two languages, you can create beautiful, functional web pages that are a joy to use.
